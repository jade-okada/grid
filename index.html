Display:Flexbox

The option "flex" must be inserted in a father-tag with elements inside it. The you can choose the direction addins the property "flex-direction" and choose if you want the "row"(line) or "column"(block). Flexbox allows you to set position, align, direction and order os the elements, all at the same time but you a single item(some especific son-tag) as well.
